# IA Business NetSolutions

Infrastructure autonome de bots pour générer des revenus sur Internet (vente d'emails, scraping, crypto, etc.)

## Modules

- **Agence Kays** : production d'emails temporaires
- **Vente PayPal** : génération de liens PayPal et boutique GitHub Pages
- **Cerveau quantique** : décision de vente adaptative
- **Scraping massif** : 500 agents + 300 boosters pour leads
- **CEO Ultra Stable** : supervision des portefeuilles réels (BNB + PayPal)
- **Auto‑réparation** : redémarrage automatique des bots

## Installation

```bash
git clone https://github.com/francky07/ia-business-networks.git
cd ia-business-networks
# Configurer les secrets dans ~/.env_ia_business
# Lancer avec tmux (voir documentation interne)
```
